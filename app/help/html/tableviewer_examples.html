TableViewer Example Folder

A selection of sample input files can be found in the example folder here as follows:

Contents

  • Data Files
  • DS property input files
  • See Also

Data Files

  • Sample_table.xlsx: sample spreadsheet with two worksheets. Sheet1 is the main data set and Sheet 2 contains 3 additional variables (it is the same as Sample_add_variables.xlsx).
  • Sample_text.txt: sample text import using tab separators.
  • Sample_mat_import.mat: import a dstable that has been saved as a mat file.
  • Sample_add_rows.txt: sample file to add 7 rows. The rows are unique names for the Sample_table.xlsx dataset.
  • Sample_add_variables.xlsx: sample file to add 3 variables. the number of rows matches the Sample_table.xlsx dataset.

DS property input files

DS properties define the meta-data associated with variables, rows and dimensions (see the class dsproperties in the dstoolbox for further details).

  • sample_dsproperties.m: function file used to load a set of dsproperties.
  • Sample_DSproperties_csv.txt: comma separated text file used to load a set of dsproperties.
  • Satmple_DSproperties_tab.txt: tab seperated text file used to load a set of dsproperties.

See Also
