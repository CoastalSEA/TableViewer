~readme.txt - this file explains the sample files available

Sample_add_rows.txt					file to Add rows of data to the dataset loaded using Sample_table or Sample_text
Sample_add_vars.xlsx				file to Add variables to the dataset loaded using Sample_table or Sample_text
sample_dsproperties.m				sample function to define the dsproperties for a data set. Use when loading Sample_table or Sample_text
Sample_DSproperties_csv.txt			csv format text file to define the dsproperties for a data set. Use when loading Sample_table or Sample_text
Sample_DSproperties_tabs.txt		tab format text file to define the dsproperties for a data set. Use when loading Sample_table or Sample_text
Sample_mat_2D_vector_import.mat		sample Matlab struct containing dstables of Data (2D vector data) and Images (tiff files).
Sample_mat_3D_array_import.mat		sample Matlab struct with 5 dstables containing 3D arrays. Tables constructed using test_dstoolbox to illustrate use of different data types for rows and dimensions.
Sample_mat_table_import.mat			sample Matlab table that can be imported using the Load option
Sample_table.xlsx					sample Excel table that can be imported using the Load option
Sample_text.txt						sample tab separated text file that can be imported using the Load option
Sample_type_test.xlsx				sample Excel table that uses different data types. Used to test Projects>Cases>Modify Variable Type. Data imported as text and modified to categoric, datatime, etc.
